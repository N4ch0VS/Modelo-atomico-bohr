# Modelo Atómico de Bohr - Sitio Interactivo

Este sitio web presenta el modelo atómico de Niels Bohr mediante animaciones interactivas que permiten visualizar el movimiento de los electrones y las transiciones entre niveles de energía.

## Características

- Animación 3D del átomo con órbitas electrónicas
- Interacción mediante arrastre del mouse para rotar la vista
- Click en electrones para excitarlos entre niveles de energía
- Múltiples capas de electrones con conteo configurable
- Modo claro/oscuro
- Simulación en tiempo real del modelo de Bohr

## Tecnologías

- HTML5 / CSS3
- JavaScript (React via JSX)
- Animaciones CSS y JavaScript
